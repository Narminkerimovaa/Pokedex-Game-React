# ⚡ Pokédex Battle Simulator

Tünd temalı, retro-arcade vizualına malik Pokémon komanda döyüş simulyatoru.

---

## 🎮 Oyun Haqqında

İki komanda arasında avtomatik döyüş simulyasiyası aparılır. Hər komandaya təsadüfi Pokémonlar bölünür və ümumi **Base Experience (XP)** xalı yüksək olan komanda qalib gəlir.

Yeni döyüş başlatmaq üçün **NEW BATTLE** düyməsini sıxmaq kifayətdir — qalanı oyun özü edir.

## ✨ Xüsusiyyətlər

- **Avtomatik bölgü** — 8 Pokémon hər döyüşdə təsadüfi olaraq iki komandaya bölünür
- **XP Sistemi** — Qalib komanda daha yüksək ümumi XP xalına sahib olan tərəfdir
- **Konfeti effekti** — Hər yeni döyüşdə partlayış animasiyası
- **Pokémon detail modal** — Karta klikləyərək Pokémon haqqında ətraflı məlumat görmək mümkündür
- **Battle History** — Keçmiş döyüşlərin vaxt, tarix və xal məlumatları sidebar-da saxlanılır
